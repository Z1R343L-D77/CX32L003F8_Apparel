# CX32L003F8 数码管显示控制器

基于 CX32L003F8（ARM Cortex-M0）单片机的双排三位数码管显示控制系统，适用于工业仪表面板场景。

---

## 硬件平台

| 项目 | 描述 |
|------|------|
| MCU | CX32L003F8，ARM Cortex-M0，4 MHz |
| 显示驱动 | FD612 LED 驱动芯片（I²C 协议） |
| 数码管 | 两排各三位七段数码管（实际可点亮三个） |
| 按键 | 三个独立按键，经由 FD612 读取 |
| 参数存储 | 片内 Flash |
| 仿真接口 | SWD，面板由仿真接口提供 5 V 电源 |
| 开发工具 | Keil MDK |

---

## 面板说明

```
┌─────────────────────────────┐
│  PV:  [ d2 ][ d1 ][ d0 ]   │  ← 上排：测量值 / 参数名称
│  SV:  [ d2 ][ d1 ][ d0 ]   │  ← 下排：设定值 / 参数数值
│  [SET]      [DEC]    [ADD]  │  ← 三个按键
└─────────────────────────────┘
```

- **PV**（上排）：显示测量信息或参数名称
- **SV**（下排）：显示设定值或参数数值
- **SET**（左键）：确认 / 进入编辑 / 长按切换模式
- **DEC**（中键）：减值 / 切换到上一个参数
- **ADD**（右键）：加值 / 切换到下一个参数

## 开机动画

上电后依次显示以下画面（每帧约 500 ms）：

| 上排 PV | 下排 SV | 说明 |
|---------|---------|------|
| `88.8` | `88.8` | 全亮自检 |
| `HB ` | `539` | 品牌 / 型号 |
| `999` | `-99` | 软件版本 |
| `G69` | `000` | 配置信息 |

## 菜单逻辑

### 默认界面

| 显示 | 说明 |
|------|------|
| 上排 `ER.1` | 测量信息 |
| 下排 当前 SV 值 | 设定值（默认 33，范围 0~255） |

- **单击 ADD / DEC**：SV 值 ±1
- **长按 ADD / DEC**（≥ 0.8 s）：SV 值快速 ±1
- **单击 SET**：切换 `ER.1` / `SV` 显示模式（切回时自动保存）
- **长按 SET**（≥ 3 s）：进入参数界面

---

### 参数界面

上排显示参数名称，下排显示参数当前值，按 DEC / ADD 在参数列表中导航：

| 参数名 | 说明 | 范围 |
|--------|------|------|
| PV | 测量值参数 | 0~255 |
| LV | 电流参数 | 0~255 |
| DV | 偏差参数 | 0~255 |
| HC | 加热控制 | 0~255 |
| IR | 积分时间 | 0~255 |
| SP | 设定点 | 0~255 |
| PI | 比例积分 | 0~255 |
| AH | 报警上限 | 0~255 |
| RV | 量程参数 | 0~255 |
| BT | 偏置温度 | 0~255 |
| INV | 反向控制 | 0~255 |
| ATV | 自整定 | 0~255 |
| **LKV** | **锁定控制**（0=只读，1=可写） | 0~1 |

> **注意**：当 LKV = 0 时，参数界面为只读模式，按 SET 无效；将 LKV 改为 1 后其余参数变为可写（默认值为 1）。

**参数界面操作：**

- **DEC / ADD**：切换到上一个 / 下一个参数
- **单击 SET**：进入编辑界面（仅 LKV=1 时有效，LKV 参数本身始终可编辑）
- **长按 SET**（≥ 3 s）：退回默认界面

---

### 编辑界面

当前参数的 SV 区域闪烁提示正在编辑：

- **ADD / DEC**：数值 ±1
- **长按 ADD / DEC**（≥ 0.8 s）：数值快速 ±1
- **单击 SET**：保存并退回参数界面
- **长按 SET**（≥ 3 s）：**不保存**，恢复原值并退回参数界面

## 工程结构

```
smg_cx32/
├── APP/
│   ├── app_menu.c/h        # 菜单状态机与参数表
│   └── app_scheduler.c/h   # 轻量级时间片调度器
├── BSP/
│   ├── bsp_flash.c/h       # Flash 参数存取
│   ├── bsp_key.c/h         # 按键事件处理（MultiButton）
│   ├── bsp_led.c/h         # LED 指示灯
│   └── bsp_smg.c/h         # 数码管显示缓冲与闪烁
├── DRIVERS/
│   ├── drv_fd612.C/H       # FD612 底层驱动
│   ├── drv_gpio.c/h        # GPIO 基础驱动
│   └── drv_key.c/h         # MultiButton 库
├── FWLB/                   # CX32L003 固件库（inc + src）
├── DEVICE/                 # 启动文件、系统头文件、Flash 算法
├── SYSTEM/
│   ├── main.c              # 入口：初始化 + 调度器主循环
│   ├── system_gpio.c/h     # 系统级 GPIO 初始化
│   └── system_it.c/h       # 中断服务（定时器 Tick）
├── MDK/                    # Keil 工程文件
└── 内容.png                # 题目原文
```

## 调度器任务

| 任务函数 | 周期 | 说明 |
|----------|------|------|
| `button_ticks` | 5 ms | MultiButton 驱动时基 |
| `LED_Task_Proc` | 10 ms | LED 闪烁处理 |
| `task_smg_scan_proc` | 1 ms | FD612 刷新 + 闪烁计数 |
| `task_app_menu_proc` | 10 ms | 菜单显示刷新 |

## 构建与烧录

1. 使用 Keil MDK 打开 `MDK/smg.uvprojx`
2. 编译（Build）
3. 通过 SWD 仿真器连接面板烧录
4. 面板由仿真接口提供 5 V 电源，无需额外供电
